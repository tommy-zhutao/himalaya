# AI News Hub 完善任务

## 背景
AI News Hub 是一个微服务架构的 AI 新闻聚合平台，目前主要功能已基本完成，本文档记录需要完善的功能点。

**更新时间：** 2026-03-14

---

## 🎯 待完成功能清单

### 1. 相关新闻推荐 ✅ 已完成（2026-03-14）
- **后端**: News API 添加相关新闻推荐接口
- **算法**: 基于 keywords 相似度（Jaccard）+ category 匹配 + source 匹配
- **评分权重**: 关键词 0.5 + 分类 0.3 + 来源 0.2
- **进度**: ✅ 完成

### 2. 用户个性化推荐 ✅ 已完成（2026-03-14）
- **后端**: 添加阅读历史表 `user_read_history`
- **算法**: 基于阅读历史 + 收藏（3倍权重）关键词匹配
- **评分**: 关键词 0.5 + 分类 0.3 + 来源 0.2 + 质量/时效加成
- **前端**: 首页"为你推荐"区块（仅登录用户可见）
- **API**: 
  - `GET /api/news/recommendations` - 获取个性化推荐
  - `POST /api/news/:id/read` - 记录阅读历史
- **进度**: ✅ 完成

### 3. 热点话题聚类 ✅ 已完成（2026-03-14）
- **后端**: `GET /api/news/trending-topics` 接口
- **算法**: 关键词词频统计 + 浏览量加权
- **前端**: 首页热点话题展示组件
- **效果**: 展示近3天热门关键词，点击跳转搜索
- **进度**: ✅ 完成

### 4. 重复检测 ✅ 已完成（2026-03-14）
- **算法**: 标题词频重叠度（Jaccard 相似度）
- **阈值**: 同来源 60%，跨来源 70%
- **范围**: 检测最近 7 天的 500 条新闻
- **效果**: RSS/API Fetcher 抓取时自动跳过重复内容
- **进度**: ✅ 完成

### 5. 用户收藏功能 ✅ 已完成（2026-03-14）
- **前端**: 用户收藏页面 `/favorites` ✅
- **前端**: 新闻详情页添加收藏按钮 ✅
- **后端**: User API 收藏接口完善 ✅
- **新增**: 添加 `/api/users/favorites/check/:id` 接口
- **状态管理**: Zustand store 管理收藏状态 ✅

### 6. 用户设置页面 ✅ 已完成（2026-03-14）
- **前端**: `/settings` 页面已完善
- **功能**: 用户名修改、头像上传、密码修改
- **后端**: User API `/api/users/me` PUT 支持密码修改
- **进度**: ✅ 完成

### 7. 管理后台增强 ✅ 已完成（2026-03-14）
