## ⚠️ 发现的问题

### 1. 搜索功能 URL 编码问题
**问题描述：**
- 使用中文关键词搜索时返回 400 错误
- 可能是 Express 路由对中文 URL 的解析问题

**复现步骤：**
```bash
curl "http://localhost:4001/api/news/search?q=36氪"
# 返回 400 Bad Request
```

**临时解决方案：**
- 使用英文关键词搜索正常工作
- 前端可以使用 encodeURIComponent 编码

**优先级：** P1（中优先级）
**预计修复时间：** 30 分钟
